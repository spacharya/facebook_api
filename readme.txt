Read Me:

For this to run download the code.
Go to https://developers.facebook.com/tools/explorer and click on Get Access Token.
Go to Advanced Publish OPtions and choose Publish.
Copy the token generated.
Generate a file called message.txt which will be used to read the messages to be posted. Using multiple messages will cause random messages to be popped out.

run the code as :
python facebook_like_comment.py

Enter the code when prompted
Enter the date when asked till which date you want to comment and like.
